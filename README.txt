Pour lancer les applications, effectuer les commandes suivantes dans l'ordre :

(make clean au cas où des fichiers subsistent)
make


1er terminal : ./serveur PORT
2ème terminal : ./client IP PORT_SERVEUR
3ème terminal : ./client IP PORT_SERVEUR

par exemple :
./serveur 12000
./client 127.0.0.1 12000
./client 127.0.0.1 12000

Pour changer d'outils (ou de pouvoirs) utilisez les flèches de haut et bas.
Pour poser un hammecons ou un pouvoir, utilisez clic gauche dans la fenêtre de simulation
Appuyez sur F2 ou CTRL+C pour quitter
